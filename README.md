# Student Management System (C)

## 📌 Description
A console-based application developed in C to manage student records using CRUD operations (Create, Read, Update, Delete).

## 🚀 Features
- Add new student records  
- View existing student details  
- Update student information  
- Delete student records  

## 🛠️ Technologies Used
- C Programming  
- Data Structures  
- Functions  

## ▶️ How to Run
1. Compile the program using a C compiler (GCC)
2. Run the executable file
3. Use the menu-driven interface to perform operations

## 🎯 Purpose
This project was developed to strengthen programming fundamentals, data handling, and problem-solving skills using structured programming.
